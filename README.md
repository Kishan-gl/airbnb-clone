# Airbnb Clone

This is a fully responsive **Airbnb UI clone** built using **HTML and CSS**.  
The clone recreates the look and feel of Airbnb's homepage with clean design and responsiveness.

## Features
- Fully responsive layout (mobile, tablet, desktop)
- Navigation bar
- Hero section with search bar
- Popular places section
- Listing cards UI
- User review section (currently in progress)

## Tech Used
- **HTML5**
- **CSS3**
- (JavaScript coming soon for interactive features)

## Currently Working On
- User review UI  
- Adding JavaScript for interactions  

## Goal
To practice frontend development and build a modern, responsive UI project.
